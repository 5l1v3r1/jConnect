jConnect
====================
Check service on target

# Installation
- `$ git clone https://github.com/j3ssie/jConnect.git `
- `$ pip install pysmb`
- `$ cd jConnect/`
- `$ python jConnect.py`

# How to use
```$python jConnect.py --target="14.171.21.132/24" --port="445" --write="test" ```
```$python jConnect.py --raw_input="data.txt" --port="445" --write="data" ```
